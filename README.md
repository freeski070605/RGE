# ReemTeam HQ

ReemTeam HQ is the private backend/admin command center for the ReemTeam ecosystem.

RGE now means **RGE Growth Engine** inside HQ. It is the growth brain that converts gameplay, users, tables, cribs, events, referrals, campaigns, and content performance into **Growth Plays**.

This repository keeps the work inside the existing `RGE` git repo. The previous implementation can still inform migration details, but the target product language is HQ-native:

- Command Center for the main dashboard
- Game Intelligence for raw activity turned into meaningful signals
- RGE Growth Engine for scoring and recommending Growth Plays
- Content Studio for content creation, review, approval, and publishing

See [docs/REEMTEAM_HQ_ARCHITECTURE.md](docs/REEMTEAM_HQ_ARCHITECTURE.md) for the platform blueprint.
